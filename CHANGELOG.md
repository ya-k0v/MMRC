# Журнал изменений MMRC

---

## [3.2.1] — 2026-04-29

### Добавлено
- **yt-dlp автозагрузка** — автоматическая загрузка бинарника yt-dlp с GitHub для работы с видео
- **Self-update система** — обновление из git с проверкой fast-forward через админ-панель
- **Миграционная система** — file-based миграции с таблицей `schema_migrations`
- **ADB автоустановка APK** — удалённая установка Android приложения через WiFi с автонастройкой
- **Android App Launcher** — запуск активностей на Android устройствах по IP
- **Кастомные страницы ошибок** — 403, 404, maintenance
- **Tablet responsive** — улучшения адаптивного дизайна для планшетов

### Изменено
- **Структура стримов** — переработан stream-manager с circuit breaker и health check
- **Управление ресурсами** — новый JobResourceManager с CPU/memory бюджетированием
- **Система уведомлений** — расширена новыми типами (stream_worker, project_update_available)
- **Ночная оптимизация** — добавлены отменяемые задачи и улучшено управление ресурсами

### Исправлено
- Обработка зависших ffmpeg процессов
- Очистка стримов при заполнении диска
- Гонки при переключении контента

## [3.2.0] — 2026-04-02

### Кратко
- Релиз: `v3.2.0`
- Кодовое имя: MMRC rebrand
- Область: ребрендинг, улучшения превью/трейлеров, исправления сборки Android

### Добавлено
- Генератор трейлеров: `src/video/trailer-generator.js` — ленивое создание ~10-секундных MP4-трейлеров
- Эндпоинт трейлера: `GET /api/files/trailer/:deviceId/:fileName`
- Улучшенное превью: Range-запросы для локальных файлов, ffmpeg для стримов
- Значок-оверлей на плеерах: `ID: <device> | v3.2.0`

### Изменено
- Полный ребрендинг: "VideoControl" → "MMRC"
- Версия увеличена с `3.0.0` → `3.2.0`
- Переработана логика превью в `player-videojs.js` и `speaker.js`
- Обработка метаданных теперь запускает генерацию трейлеров

### Исправлено
- Сборка Android: устранена ошибка `Unresolved reference: BuildConfig`
- Черный экран в превью на панели спикера
- Статические превью больше не загружаются в Video.js
- Устранены гонки при переключении плейсхолдера и контента

### Безопасность и стабильность
- Обработчики превью выставляют `X-Accel-Buffering: no`
- Генерация превью для стримов использует консервативную ffmpeg команду

## [3.0.0] — 2025-XX-XX

### Добавлено
- **HLS/DASH стриминг** — live streaming через FFmpeg
- **Ночная оптимизация** — автоматическая оптимизация видео в ночные часы
- **Профили кодирования** — 720p/1080p/2160p с настраиваемыми параметрами
- **LDAP/Active Directory** — корпоративная аутентификация
- **Система уведомлений** — критические/предупреждения/инфо уведомления
- **Circuit Breaker** — защита БД, файловой системы и внешних API
- **Audit Log** — полный журнал действий
- **Metrics** — метрики запросов, БД, Socket.IO
- **Hero Module** — картотека героев с отдельной БД и панелями
- **PWA** — прогрессивные веб-приложения для панелей
- **Тёмная/светлая тема** — переключение во всех панелях
- **MPV Client** — нативный плеер для Linux с аппаратным ускорением
- **Server-side плейлисты** — автолупинг папок без участия клиента
- **Частичный MD5** — дедупликация для файлов >100MB
- **Транслитерация** — кириллица в именах файлов
- **MIME-валидация** — проверка типов файлов
- **Rate Limiting** — с пропуском для локальных IP
- **CSRF защита**
- **Request Timeout**
- **Resolution Cache** — LRU кэш для разрешений видео
- **MP4 Faststart** — обработка перед дедупликацией
- **Winston Logging** — с ротацией логов
- **System Monitor** — мониторинг диска, БД, памяти, ffmpeg
- **Timer Registry** — централизованное управление таймерами

### Изменено
- Переработана архитектура Socket.IO (connection manager, handlers)
- Улучшена система загрузки файлов
- Расширена админ-панель (стримы, настройки)
- Переработана спикер-панель

### Исправлено
- Множество багов в обработке видео
- Проблемы с памятью при обработке больших файлов
- Проблемы с подключением Android клиентов
